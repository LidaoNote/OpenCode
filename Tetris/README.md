# Premium Tetris (Global Professional Edition)

这是一个基于 Python 和 Pyglet 构建的、完全遵循国际竞赛标准的顶级俄罗斯方块实现。

## 🌟 技术亮点 (KISS 原则)

1.  **7-Bag 随机算法**：采用竞赛标准，每 7 个方块为一组（包含所有种类）打乱发放。告别“脸黑”，保证极致的公平性。
2.  **现代锁定延迟 (Lock Delay)**：方块落地后拥有 0.5 秒的机动缓冲时间。支持移动/旋转重置计时（最高 15 次），完美支持塔尖微操。
3.  **SRS 踢墙补偿 (Wall Kicks)**：优化的旋转检测系统，在贴墙或狭窄处旋转时自动寻找合法位置，手感丝滑顺畅。
4.  **T-Spin 检测与高额奖励**：完整支持 T-Spin 检测（3-Corner 规则），并为 T-Spin Single/Double/Triple 提供竞赛级的得分倍率。
5.  **工业级输入处理 (DAS/ARR)**：支持自定义的延迟自动重复（DAS）和自动重复频率（ARR），手感反馈达到毫秒级响应。
6.  **全平台手柄原生适配**：支持 Xbox/PS 等主流手柄，带有一键暂停、一键重置等快捷操作。
7.  **自给自足的音频引擎**：内置全代码合成的音频系统，支持正弦波、方波和白噪声，无需任何外部音频资源文件。

## 🚀 快乐开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行游戏
```bash
python tetris.py
```

## 🎮 操作说明

| 动作 | 键盘按键 | 手柄按键 |
| :--- | :--- | :--- |
| **移动** | 方向键 左/右/下 | 左摇杆 / 十字键 |
| **快速硬降** | 空格 (SPACE) | 按钮 Y / X |
| **旋转** | 方向键 上 (UP) | A 按钮 / 十字键上 |
| **暂存 (Hold)** | C / Shift | L1 / R1 |
| **暂停/继续** | P | Start 键 |
| **重置游戏** | R | Select 键 |
